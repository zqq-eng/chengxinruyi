《秤心如意》小程序示例工程（精简可运行版）

1. 使用微信开发者工具打开本目录（包含 app.json 的这一层）。
2. 在 app.js 中把 env 替换为你的云开发环境 ID。
3. 在 cloudfunctions/login、ai_analyze、batch_import 右键“上传并部署：云端安装依赖”。
4. 在云开发数据库中创建集合：users、weights 等。
5. 在环境变量中配置 OPENAI_API_KEY（可选，用于 AI 分析）。

本工程包含：
- 登录 / 注册（示例）
- 首页（今日体重 + 快捷入口）
- 记录体重页面
- 社区 / 预约 / 批量导入等占位页面
- 粒子背景组件
- 基本云函数结构
