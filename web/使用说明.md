# Web后台管理系统使用说明

## 当前环境问题

当前环境缺少必要的运行时环境（Python或Node.js），无法自动启动代理服务。

## 解决方案

### 方案一：使用Python（推荐）

1. 确保安装了Python 3.7+
2. 安装Flask和requests：
   ```
   pip install flask requests
   ```
3. 配置微信云开发参数：
   - 打开 `proxy.py` 文件
   - 修改 `WECHAT_CONFIG` 中的 `APPID` 和 `SECRET` 为你的小程序实际值

4. 启动服务：
   ```
   cd web
   python proxy.py
   ```

5. 访问 http://localhost:3000/

### 方案二：使用Node.js

1. 确保安装了Node.js 14+
2. 安装依赖：
   ```
   cd web
   npm install
   ```
3. 配置微信云开发参数：
   - 打开 `server.js` 文件
   - 修改 `WECHAT_CONFIG` 中的 `APPID` 和 `SECRET` 为你的小程序实际值

4. 启动服务：
   ```
   npm start
   ```

5. 访问 http://localhost:3000/

### 方案三：手动部署到云服务器

如果你有云服务器（阿里云、腾讯云等），可以在服务器上部署代理服务，然后修改前端的API地址为你的服务器地址。

## 重要配置

无论使用哪种方案，都需要配置微信云开发参数：

```python
# proxy.py 中的配置
WECHAT_CONFIG = {
    'APPID': '你的小程序AppID',
    'SECRET': '你的小程序AppSecret',
    'CLOUD_FUNCTION_NAME': 'admin_api'
}
```

```javascript
// server.js 中的配置
const WECHAT_CONFIG = {
    APPID: '你的小程序AppID',
    SECRET: '你的小程序AppSecret',
    CLOUD_FUNCTION_URL: 'https://api.weixin.qq.com/tcb/invokecloudfunction'
};
```

## 获取微信云开发参数

1. 登录微信公众平台：https://mp.weixin.qq.com/
2. 进入小程序后台
3. 点击「开发」→「开发管理」→「开发设置」
4. 获取 AppID 和 AppSecret

- 控制台：数据统计概览
- 用户管理：用户列表和编辑
